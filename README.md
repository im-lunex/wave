# Wave Shell

This project is a functional shell implementation that handles core builtins like `exit`, `echo`, `type`, and `pwd`, alongside a navigation system supporting absolute paths, relative paths, and tilde expansion for the home directory. It also includes an external command runner that searches the system `PATH` to execute binaries and employs a custom splitting logic to process user input.
